Detailed Description

This class represents the minimum price increment for a price range.

A PriceIncrRow object is used to indicate a minimum price increment for a range of prices. The range is defined by an upper and/or lower bound on the price. A value of CmpOperator::None for an operator means that there is no constraint. When the operator is CmpOperator::None, the corresponding price is not valid and should not be examined.

Therefore, if both operators are CmpOperator::None, there are no constraints on the price range, and the minimum price increment is constant for all prices.

For example, if the minimum price increment is 0.25 for all prices, the instance will have the following values :

  • PriceIncrRow::PriceIncr = 0.25
  • PriceIncrRow::Precision = 2
  • PriceIncrRow::FirstOperator = CmpOperator::None
  • PriceIncrRow::LastOperator = CmpOperator::None

If one operator is CmpOperator::None and the other is not, there is a single constraint.

For example, if the minimum price increment is 0.10 when the price of the instrument falls below -3.00, then the instance will have the following values :

  • PriceIncrRow::PriceIncr = 0.10
  • PriceIncrRow::Precision = 2
  • PriceIncrRow::FirstOperator = CmpOperator::LessThan
  • PriceIncrRow::FirstPrice = -3.00
  • PriceIncrRow::LastOperator = CmpOperator::None

Another example is if the minimum price increment is 0.10 when the price of the instrument is above 3.00, the instance may have the following values :

  • PriceIncrRow::PriceIncr = 0.10
  • PriceIncrRow::Precision = 2
  • PriceIncrRow::FirstOperator = CmpOperator::None
  • PriceIncrRow::LastOperator = CmpOperator::GreaterThan
  • PriceIncrRow::LastPrice = 3.0

Finally, if both operators are not CmpOperator::None, then there are two constraints that apply to the current price increment.

For example, if the minimum price increment is 0.05 when the price is greater than or equal to -3.00 and less than or equal to 3.00, the instance may have the following values :

  • PriceIncrRow::PriceIncr = 0.05
  • PriceIncrRow::Precision = 2
  • PriceIncrRow::FirstOperator = CmpOperator::GreaterThanOrEqualTo
  • PriceIncrRow::FirstPrice = -3.00
  • PriceIncrRow::LastOperator = CmpOperator::LessThanOrEqualTo
  • PriceIncrRow::LastPrice = 3.00

◆ Tag

string Tag
getset

This optional value specifies a tag that is persisted with the resultant order by the infrastructure.

If specified, this value will be present on subsequent order updates published by the infrastructure.

If this value is empty or null, no tag will be set.

This value differs from OrderParams.UserMsg in that it cannot be updated with subsequent order operations.

This value differs from OrderParams.UserTag in that this value is used by R | Trader.

◆ UserMsg

string UserMsg
getset

This optional value specifies a user message that is persisted with the resultant order by the infrastructure.

If specified, this value will be present on subsequent order updates published by the infrastructure.

If this value is empty or null, no user msg will be set.

Note : The value can be updated with any subsequent order operation including other instances of R | API and R | Trader.

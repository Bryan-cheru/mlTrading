Detailed Description

This class represents market mode (trading status) information of an instrument.

This class represents market mode (trading status) information of an instrument received as a result of calls to REngine::subscribe() via RCallbacks::MarketMode().

1, 1970) as presented by the exchange/market data source. Generally this timestamp correlates to the event time, rather than a transmission (JOP) or receipt timestamp.

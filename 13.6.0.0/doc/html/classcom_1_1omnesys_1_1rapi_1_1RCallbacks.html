Detailed Description

This virtual class contains all the user-defined callbacks for R | API.NET.

A subclass of RCallbacks must be defined, instantiated and passed into REngine::login() when logging into the infrastructure. While the session is active, the instance passed in as the RCallbacks parameter must not be destroyed.

◆ LimitOrderBook()

virtual void LimitOrderBook ( OrderBookInfo  oInfo)
inlinevirtual

This method is invoked when the engine receives a limit order book.

The limit order book may be the result of a subscription to MD_QUOTES, or a call to REngine::rebuildBook.

If the limit order book is due to a subscription to MD_QUOTES, the callback represents a snapshot of the order book against which subsequent quotes can be applied. In other words, the RCallbacks::LimitOrderBook method provides an initial order book state, and RCallbacks::AskQuote and RCallbacks::BidQuote provide updates to that initial book.

This callback can also indicate the clearing of one or both sides of the order book, in which a case the number of bids and/or the number of asks is zero.

Parameters
oInfothe LimitOrderBookInfo object that contains information about the order book.
See also
REngine::rebuildBook()
REngine::subscribe()
SubscriptionFlags::Quotes

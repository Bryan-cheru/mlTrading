◆ UserMsg

string UserMsg
getset

This optional value specifies a user message that is persisted with the resultant quote by the infrastructure.

If specified, this value will be present on subsequent quote updates published by the infrastructure.

If this value is empty, no user msg will be set.

Note : The value can be updated with any subsequent operation on the quote, including other instances of R | API and R | Trader.

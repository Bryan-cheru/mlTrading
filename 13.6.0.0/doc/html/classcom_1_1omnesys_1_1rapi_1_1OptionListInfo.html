Detailed Description

An instance of this class is the response to a call to REngine::getOptionList().

An OptionListInfo object is used to pass :

  • a list of exchanges,
  • a list of product codes within an exchange, or
  • a list of expiration dates within a product code, or
  • a list of RefDataInfo objects representing options with an expiration date within an underlying.

What is returned depends upon what was initially passed into REngine::getOptionList().

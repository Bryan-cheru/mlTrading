Detailed Description

This class represents close mid price information.

The CloseMidPriceInfo class is used to convey the close mid price for a trading session.

To express interest in mid price data, one can specify SubscriptionFlags.MidPrice when subscribing. The MidPriceInfo class is used to pass mid price info via RCallbacks::MidPrice() and the CloseMidPriceInfo class is used to convey close mid prices via RCallbacks::CloseMidPrice().

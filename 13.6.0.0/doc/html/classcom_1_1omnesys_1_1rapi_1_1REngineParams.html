Detailed Description

This class contains the parameters for an REngine instance.

Many of these values will be automatically inserted in the default environment keyed by "system".

See also
REngine::listEnvironments()

◆ LogFilePath

string LogFilePath
getset

Log file path.

Optional. If this value is specified, the R | API will create log files based on the log file path name. They will have numeric extensions beginning with *.000, and will be incremented as the log files are rolled over. Log files are normally rolled over when they reach a size of around 500mb.

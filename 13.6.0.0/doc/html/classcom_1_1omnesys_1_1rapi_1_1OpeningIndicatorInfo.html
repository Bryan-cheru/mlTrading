Detailed Description

This class represents opening indicators.

The OpeningIndicatorInfo class is used to pass opening indicator price information received as a result of calls to REngine::subscribe() via RCallbacks::OpeningIndicator(). Not all exchanges or instruments make use of opening indicators.

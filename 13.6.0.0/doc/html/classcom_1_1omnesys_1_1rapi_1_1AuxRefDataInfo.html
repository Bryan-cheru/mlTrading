Detailed Description

This class represents auxiliary reference data for an instrument.

The AuxRefDataInfo class is used to pass auxiliary reference data resulting from calls to REngine::getAuxRefData() or REngine::subscribe() where the flags include SubscriptionFlags.RefData. The data is passed via the callback RCallbacks::AuxRefData(). Please note that this auxiliary reference data is seperate and distinct from RefDataInfo and PriceIncrInfo.

◆ IsTradable

string IsTradable
getset

If specified, indicates whether this instrument is tradable or not.

Expected values are "Y", "N", or empty. Generally, if the value is empty, the instrument is tradable.

Sometimes an instrument exists but is not tradable. For example, there is a case where an option's underlying does not actually exist as a tradable instrument.

double SizeMultiplier

When not NaN, this value can be used to multiply sizes/quantities/volumes associated with this instrument for display purposes.

Detailed Description

An instance of this class represents a change to an account.

The AccountUpdateInfo class conveys that some update has happened to the Account specified by AccountUpdateInfo::Account. The nature of the update is indicated by what action (AccountUpdateInfo::Action) has occurred.

See also
RCallbacks::AccountUpdate().

Detailed Description

This class represents the end to a set of updates to an order book.

The EndQuoteInfo class is conveys that a set of quote updates to the instrument's order book is over, and the order book can be evaluated. This class is similar to the AskInfo or BidInfo classes where AskInfo.UpdateType and/or BidInfo.UpdateType is UpdateType.End, except that this class does not include price level information.

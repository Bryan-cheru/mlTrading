Detailed Description

This class represents information about quotes.

The QuoteInfo class is used to pass quote information (including various states of the quote) via RCallbacks::Quote().

Indicates whether the AskPriceToFill and AskQuantityToTill contain values.

It may be possible to receive updates on a quote which do not specify (i.e. reiterate) the bid/ask prices and sizes. In these cases, this flag may be false even though the quote is open.

Detailed Description

This class represents the definition of a multi-leg strategy (spread).

A StrategyInfo object is used to present a multi-leg spread, with the ratios of each leg. The legs are presented as a collection of StrategyLegInfos. This information is retrieved using REngine::getStrategyInfo().
